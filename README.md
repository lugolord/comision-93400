# Ecommerce Mercatto

Aplicación de ecommerce desarrollada con React que permite explorar productos, filtrarlos por categoría, visualizar información detallada, gestionar un carrito de compras y finalizar órdenes de compra.

## 🚀 Funcionalidades

* Listado de productos.
* Filtrado de productos por categoría.
* Visualización del detalle de cada producto.
* Gestión del carrito de compras.
* Finalización de órdenes de compra.
* Persistencia de datos mediante Firebase.

## 🛠️ Tecnologías utilizadas

* React
* Tailwind CSS
* DaisyUI
* React Router
* Firebase

## 📦 Dependencias principales

* [Tailwind CSS](https://tailwindcss.com/docs)
* [DaisyUI](https://daisyui.com/docs/)
* [React Router](https://reactrouter.com/home)
* [Firebase](https://firebase.google.com/docs)
